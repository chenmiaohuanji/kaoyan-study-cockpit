# 研途驾驶舱

面向长期备战考研用户的个人学习控制台。项目把目标、计划、今日执行、专注计时、科目进度、错题、成绩、周期报告与数学建模训练放在同一套安静、克制的界面中。

## 已实现

- 首页总览：核心指标、今日任务、科目推进、学习趋势与风险提醒
- 今日学习：筛选、排序、拖拽、临时任务、延期原因、完成记录与专注模式
- 学习计划：阶段、月、周、日四级视图，可新增计划并同步到今日任务
- 科目进度：可录入学习记录，联动更新章节掌握度、科目进度和学习时长
- 错题本：可新增、筛选、删除，并在单题复习模式中记录掌握反馈
- 测试成绩：可录入和删除成绩，实时重算趋势、预测分与目标差距
- 学习报告：日报、周报、月报、阶段报告与 Markdown 导出
- 数学建模：独立统计、时间预算、团队任务新增与状态推进
- 深色模式、减少动画、桌面/平板/移动端布局
- 浏览器本地保存、JSON 数据导入与导出

## 本地开发

```bash
npm install
npm run dev
```

常用命令：

```bash
npm run build
npm run build:pages
npm test
npm run lint
```

## 部署

`npm run build:pages` 生成 GitHub Pages 静态产物。线上版本不要求账号验证，学习记录仅保存在当前浏览器的 `localStorage` 中。

完整视觉与交互规范见 [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md)。
